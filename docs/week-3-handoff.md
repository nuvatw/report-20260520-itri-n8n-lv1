# 第三週交付紀錄

## 交付目標

第三週目標是強化首頁瀏覽體驗，讓報告數量增加後仍能透過搜尋、分類、年度、月份與最新標示快速找到成果報告。

## 已完成範圍

- 首頁新增年度篩選 chips。
- 首頁新增月份索引列。
- 分類與年度 chips 顯示報告數量。
- 首頁新增月份總數統計。
- 首頁新增 active filter tokens。
- 首頁新增清除條件按鈕。
- 最新報告卡片新增 `Latest` badge 與醒目框線。
- 空狀態拆成索引缺失、無報告、無符合條件三種情境。
- 建立首頁瀏覽體驗標準：`docs/homepage-browsing-standard.md`。
- 建立第三週驗收腳本：`scripts/verify-week-three.mjs`。
- 將第三週完整驗收保留為 `npm run verify:week3`。

## 第三週驗收清單

- 首頁顯示兩份既有報告。
- 搜尋「工研院」只顯示 n8n 課程報告。
- 分類「活動成果」只顯示新北站報告。
- 年度 `2026` 可被選取。
- 月份 `2026.05` 可被選取。
- 最新報告顯示 `Latest`。
- 啟用條件後側欄顯示 token 與清除按鈕。
- 行動版報告卡片仍可閱讀，且開啟報告不換行。
- `npm run build`、`npm run verify:week1`、`npm run verify:week2`、`npm run verify:week3` 均通過。

## 下一週入口

第四週可以進入發布與驗收流程：加入 build 檢查、連結抽查、行動版視覺檢查與靜態部署流程，確保新增報告不破壞首頁。
